# City Go — Технический долг

Источник истины для технического долга проекта City Go.

## Правила ведения

- Новые пункты добавлять в конец соответствующего раздела.
- Не удалять пункт без явного решения.
- После закрытия менять статус на `Done`.
- Если пункт перенесен в задачу/issue, добавлять ссылку или номер задачи.

## Статусы

- `Open` — актуально, не взято в работу.
- `In Progress` — взято в работу.
- `Done` — закрыто.
- `Deferred` — отложено.
- `Rejected` — отменено.
