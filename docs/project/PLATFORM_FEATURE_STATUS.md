## Текущее состояние проекта

### Сильные стороны
- маршрутный движок;
- импорт данных;
- обогащение данных;
- Eligibility Dashboard;
- Data Quality;
- City Readiness;
- Dry Run.

### Основные проблемы
- неполные данные;
- качество маршрутов нестабильно;
- слабый пользовательский слой;
- карта требует переработки;
- отсутствуют профили пользователей;
- отсутствует монетизация.
